El Path Tracing es un método de Monte Carlo de gráficos por computadoras para el renderizado de imágenes de escenas, de tal manera que la iluminación global sea fiel a la realidad.
Durante el curso de Análisis de Algoritmos impartido por el profesor Yuen Law durante el primer semestre del año 2020 se nos solicitó implementar una solución de Path Tracing 2D la cual debía utilizar las técnicas de algoritmos probabilísticos Monte Carlo para calcular la cantidad de rayos y la dirección de estos.
